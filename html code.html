Problem Statement

The tourism industry is dynamic, requiring insights from data for strategic decisions. Over-tourism affects cities like Barcelona and Venice. This project uses data analytics to analyze patterns and improve decision-making while promoting sustainable tourism.

Abstract

The project analyzes structured tourism data from 2014 to 2020 using tools like Python, Pandas, and Scikit-learn. Objectives include forecasting, segmentation, COVID impact, and visual dashboard deployment using Streamlit, Flask, and Gradio.
